> **Staged, gated rollout (see design D12).** This area has a history of confident-but-wrong designs (inferred ratings #1155, monolithic-D-Flow, capped-variant, envelope-first). The plan is therefore phased with hard STOP gates: prove value cheaply *before* the bulk build, and pre-commit to abandoning the speculative phases if a gate comes back flat. Do **not** skip a gate or start a later phase before the prior gate passes. `correct-dflow-aflow-editor-profile-docs` (editor-vs-profile prose + stale A-Flow names) is being implemented in parallel and is independent — both only ever *append* `Expected*:` lines / fix prose and never rename headers/aliases, so they don't collide.
