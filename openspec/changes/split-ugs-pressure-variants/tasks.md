- [ ] 3.4 Seed `Expected*:` lines on the **archetype-diverse initial 5** only — every value traceable to that profile's existing source-tagged "Official dial-in target" / "Grind diagnostics" line in `docs/PROFILE_KNOWLEDGE_BASE.md` (record the `[SRC]` line beside each, per design D12; invent nothing):
